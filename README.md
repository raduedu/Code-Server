This is my code server backup code. I usually use this for my CS tutoring when away from a computer.
Here is the basic idea which I fortunately made work: I have a CasaOS server made with an older laptop, I installed a docker container running Code Server. I am now using this Code Server and a C++ compiler to run any C++ code I would need for tutoring or other random C++ projects that would only need to be temporarily stored. This repo is my way of sharing this Code Server idea with the world showcasing that you too can program from any device from absolutely everywhere! (When not at home I usually bring my tablet, don't carry my laptop, love the VS Code interface, hate most android apps, decided to run this web-based on my own little server which uses Tailscale for it to be accessible from anywhere)
